## Keyboard shortcuts

- Ctrl-Z: undo
- D: drop to workspace
- Ctrl+D: duplicate
- Shift+rotating: rotate 45 degrees
- Shift+scaling: uniform scaling
- Alt+Shift+scaling: uniform scaling about centre
