### 🧭 Navigate the Workplane

By doing the examples on this site, you will develop the following core competencies in Tinkercad.

| Image | Tool | Actions | How to Use |
|-------|------|--------|--------------|
| <Image src={viewcube} alt="View Cube"/> | **View Cube** | 1. Rotate workplane <br/>2. Rotate to preset Views | 1. Drag cube to rotate workplane <br/>2. Click faces for **Top**, **Front**, or **Side** views. |
| <Image src={home} alt="View Cube"/> | **Home** | Reset View | Returns to default front-facing perspective. |
| <Image src={fitall} alt="View Cube"/> | **Zoom fit all** | Zoom fit selected | Fits screen (nothing selected) or centers on **selected objects**. |
| <Image src={switchview} alt="View Cube"/> | **Perspective/Flat** | Toggle View | Switches between 3D **Perspective** view and flat **Orthogonal** view. |

### ✏️ Transform

**Move**

move object by first making sure nothing is selected(click off or click blank space), then simply click and drag in one motion.

| Action | Shortcut | Result |
|--------|----------|--------|
| Horizontal | Arrow Keys | Move 1 grid unit. |
| Horizontal (Large) | `Shift` + Arrows | Move 10 grid units. |
| Vertical | Drag Up Arrow Cone | Raise or lower object. |

**Rotate**

| Action | Shortcut | Result |
|--------|----------|--------|
| Set Angle | Select Handle → `Tab` → Type # | Rotate to exact degree. |

**Resize**

| Action | Shortcut | Result |
|--------|----------|--------|
| Set Dimension | Select Handle → `Tab` → Type # | Resize to exact size. |
| Next Dimension | Press `Tab` | Jump between width/depth/height inputs. |
| Confirm | Press `Enter` | Finalize change. |
