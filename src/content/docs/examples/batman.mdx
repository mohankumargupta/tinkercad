## Steps

#### Step 1

Get batman logo SVG from [here](https://worldvectorlogo.com/logo/batman-5/)

#### Step2 

Import SVG into  Tinkercad 

<Image src={base} alt="batman svg"/>

#### Step 3

Drop in a cylinder so that it covers the logo but doesnt touch the outer ring. 
Use align to center both.

<Image src={batman2} alt="batman svg"/>

<Image src={batman3} alt="batman svg"/>

#### Step 4

Select both shapes and press group icon, then select intersect option.
