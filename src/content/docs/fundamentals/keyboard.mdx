You can get away with not using Keyboard shortcuts in Tinkercad, but some are just invaluable to make your workflow in Tinkercad faster.

## Most important keyboard shortcuts

- **Ctrl-Z**: undo
- **D**: drop to workspace
- **Ctrl+D**: duplicate
- **Shift+Right click+move mouse**: Panning
- **Ctrl+Arrow Keys**: nudge a shape vertically
- **Tab**: Give focus to entering dimensions in textbox directly on workspace when object selected. Also use tab when to gain focus to textbox where you can angle when rotating.
